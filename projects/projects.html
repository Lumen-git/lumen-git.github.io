Work & Projects

All of my main computer science (or cs adjacent) work. All tools and software I've wrote can be found at the bottom of this page or on my GitHub page. Various games and game mods I've made can be found on my games & mods page. Click on the image to go to the projects main page.


ThingsAI

ThingsAI takes a collection of smaller images and icons (things) and uses an evolution algorithm to create a dynamic mosaic of a larger input image. To the left is the Mona Lisa created using the textures from Minecraft.

The Toybox

The Toybox is a collection of small programs that experiment in computer science and art. Some use visual tools such as p5js, classic tech like C46 Basic, or esolangs. To the right is a demo of Perlin noise to generate terrain and control the motions of a ship.

Simple ECA

Simple ECA that takes a Wolfram code and a number of generations and simulates an elementary cellular automata. It then shows a matplotlib graph of all generations

DROP-ZIP-EXECUTE

DROP-ZIP-EXECUTE is a USB Rubber Ducky payload that 1) Downloads a zip file (tested with Dropbox) 2) Unzips the contents to a given folder and 3) Runs a contained file. This becomes extra powerful when the software is not executed directly, but first moved about the computer and made hidden using a Batch or Bash file, then executed to hide all tracks.

3D Garden Labels

This is a collection of SVG and Illustrator files that can be laser cut or 3D printed to create unique crop labels for your garden. (sneak attack I'm also a Botanist!)
